## Examination Types Covered

All guides cover these examination types:

### Ministry of Manpower (MOM)
- 6-Monthly MDW (Domestic Worker)
- 6-Monthly FMW (Foreign Worker)
- Work Permit Medical

### Immigration & Checkpoints Authority (ICA)
- PR Medical (Permanent Residence)
- Student Pass Medical
- LTVP Medical (Long Term Visit Pass)

### Land Transport Authority (LTA) / Traffic Police (TP)
- Driving Licence (TP) - Taxi/Private Hire
- Driving Vocational (TP/LTA) - Combined vocational license
- Vocational Licence (LTA) - LTA vocational only

### Other
- Aged Drivers Assessment
